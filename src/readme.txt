Cách chạy chương trình py:

Yêu cầu:
	Máy tính đã cài đặt python
	Máy tính đã cài đặt các thư viện cần thiết liên quan: matplotlib,...
Hướng dẫn:
1.	Mở cmd tại thư mục src
2.	Gõ python main.py <tên file test/ đường dẫn>
	Để sử dụng các file test có sẵn, ta gõ dòng lệnh sau: python main.py ../test/<tên file>
		VD: python main.py ../test/maze2.txt
	Mặc định chương trình sẽ sử dụng file maze1.txt trong thư mục ../test
3.	Kết quả hiển thị lần lượt là kích thước mê cung, kết quả các thuật toán và chi phí
